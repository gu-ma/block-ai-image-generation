# Image Generation Overview

## Intro

This block focuses on how the creative process, inspiration, and prompting evolve when working with AI image systems.

## Possible Schedule

| Time    | Activity                                      |
| ------- | --------------------------------------------- |
| 5 mins  | Touching base: current tools and expectations |
| 20 mins | Creative process with AI image generation     |
| 25 mins | Technical foundations (diffusion pipeline)    |
| 20 mins | Prompting lab (prompt + negative + weights)   |
| 10 mins | Break                                         |
| 50 mins | Research, generation sprint, and discussion   |

## Block Goals

1. Understand how AI image workflows change ideation, iteration, and authorship
2. Build practical prompting habits for better control and fewer artifacts
3. Compare model/tool choices and document trade-offs for a project context
